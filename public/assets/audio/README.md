# 🔊 Dossier Son — Guide des fichiers audio

Tous les sons de l'application Nostalgia Hub OS sont stockés ici.
Pour modifier un son, remplacez simplement le fichier par votre propre MP3 ou WAV
en gardant le **même nom de fichier**.

## Effets sonores (UI)

| Fichier | Se joue quand... |
|---------|-----------------|
| `click_survol_tuile.wav` | On **survole** une tuile de la grille |
| `pop_interaction.wav` | On **clique** sur un avatar Mii ou un thème |
| `ouverture_fenetre.wav` | On **ouvre** une fenêtre d'application |
| `fermeture_fenetre.wav` | On **ferme** une fenêtre |

## Musiques de fond

| Fichier | Description |
|---------|------------|
| `musique_plaza_morning.wav` | Musique d'ambiance #1 "Plaza Morning" — douce et calme |
| `Eshop July 2014.wav` | Musique d'ambiance #2 "Mii Channel Vibe" — ludique |
| `musique_shop_ambient.wav` | Musique d'ambiance #3 "Shop Ambient" — relaxante |

## 💡 Comment modifier les sons

1. Trouvez un fichier MP3 ou WAV que vous aimez
2. Renommez-le avec le **même nom** que le fichier à remplacer
3. Remplacez le fichier dans ce dossier
4. Rechargez la page — votre nouveau son sera utilisé !

> ⚠️ Gardez des fichiers courts pour les effets sonores (< 1 seconde).
> Les musiques de fond peuvent être aussi longues que vous le souhaitez.
